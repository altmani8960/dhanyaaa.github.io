Main Requirements

You must always have 2 main elements to organize and constitute our page. They are:

Element:!DOCTYPE

Description: It tells the computer what language is being used.
Syntax:

  • start tag:<!DOCTYPE>
  • content: language used
  • end tag: none

Element:HTML

Description: It tells the computer what program is being used.
Syntax:

  • start tag:<html>
  • content: language used
  • end tag: </html>

Head

Element:Head

Description:It keeps all of the elemnts of the header together.
Syntax:

  • start tag:<head>
  • content: header elements
  • end tag: </head>

Element:Title

Description:It is the title of the page loacted at the top of the tab.
Syntax:

  • start tag:<title>
  • content: title of the site
  • end tag: </title>

Body

Element:Body

Description:It organizes the code so that the content is easy to find.
Syntax:

  • start tag:<body>
  • content: The entire contents of the site.
  • end tag: </body>

Element:Headings

Description:This is what organizes and titles paragraphs or the entire page.
Syntax:

  • start tag:<h1>(The number can be changed depending on the size you want it.
  • content: title of the page or paragraph.
  • end tag: </h1>

Examples:

<h1>

Heading 1

<h2>

Heading 2

<h3>

Heading 3

Element:Paragraphs

Description:It is the text located in the body.
Syntax:

  • start tag:<p>
  • content: text of the site
  • end tag: </p>

Example:
The paragraph elemnt is what was used to write this text.

Element:Breaks

Description:It splits paragraphs up.
Syntax:

  • start tag:<br>
  • content: It splits paragraphs.
  • end tag: none

Example:
text<br>more text
text
more text

Element:Lines

Description:It puts lines in between paragraphs or wherever you want them.
Syntax:

  • start tag:<hr>
  • content:It makes a line between content.
  • end tag: none

Example:
text<hr>more text
text

more text

Element:Comments

Description:It lets you place comments in the code that only you can view.
Syntax:

  • start tag:<!--
  • content:It makes a comment that can only be viewed by you.
  • end tag:-->

Example:
<!--Comment-->
If you click ctrl U, you can see all of my comments I used to organize along with my message to you.

Element:Links

Description:It hyperlinks a site that you want.
Syntax:

  • start tag:<a href="url">
  • content: text that you want shown in place of the url.
  • end tag: </a>

Example:
<a href="www.google.com">A very famous search engine.</a>
A very famous search engine.

Element:Images

Description:It places an image in your site.
Syntax:

  • start tag:<img src="url">
  • content:The image will show up on your page. No content in needed unless you wanted to add an alternaive text to the image which you put alt="text" after the url. You can also adjust the size by using width="n" or height="n".
  • end tag:none

Example:
<img src="https://static.pexels.com/photos/36753/flower-purple-lical-blosso.jpg" width="55" height="48">

<img src="https://static.pexels.com/photos/36753/flower-purple-lical-blosso.jpg" width="75" height="60">

Element:Lists

Description:It makes lists that are numbered or bulleted in text.
Syntax:

  • start tag:<ul or ol>This depends on weather you want a ordered or unordered list. <li> This goes before evry listed item to recognize that it is seperate
  • content: listed items
  • end tag: </ul or ol> </li>

Example:
<ul> <li>an</li> <li>unordered</li> <li>list</li> </ul>

  • An
  • Unordered
  • List

<ol> <li>an</li> <li>ordered</li> <li>list</li> </ol>

  1. An
  2. Ordered
  3. List
